✨ 插件列表

🛡️ 敏感数据脱敏
扫描所有的会话记录（包括 Prompt 和上下文文件），寻找并替换 API Keys (OpenAI, Anthropic 等) 和 Access Keys (百度云, AWS等)。
将这些私密信息统一安全地脱敏成了 ***MASKED***，并提供脱敏统计报告。
🧑‍💼 个人隐私信息清理
自动扫描并脱敏常见的个人隐私信息（PII），包括：大陆身份证号码、银行卡号、手机号、电子邮件等。
将隐私信息替换为 ***PII_MASKED***。
🏢 商业机密与内网资产脱敏
扫描并脱敏数据库连接串（MySQL, MongoDB等）、内网 IP 以及 RSA 私钥/证书。
✂️ 冗余上下文与报错瘦身
对单条超过 3000 字符的超长消息（如冗余文件内容、超长报错堆栈）进行首尾截断，大幅减小数据体积。
🤔 废话与问候语过滤
自动去除 AI 回复中常见的客套话（如“好的，我会...”、“希望对你有帮助”等），保留核心知识。
🔠 敏感变量与 JWT 扫描
扫描代码中的硬编码密码变量（password = "..."）以及过长的 JSON Web Tokens (JWT)。
